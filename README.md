# Personal Portfolio Website

Personal portfolio website showcasing my projects, experience, and technical expertise. Built with modern web technologies and optimized for performance across all devices.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Interactive Portfolio**: Dynamic video previews with static image fallbacks
- **Performance Optimized**: Lazy loading, optimized assets, and efficient animations
- **Modern UI/UX**: Clean design with smooth transitions and professional aesthetics
- **Real-time Monitoring**: Integration with live infrastructure monitoring dashboard

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Animations**: Vanta.js for dynamic backgrounds
- **Icons**: Ionicons 7.1.0
- **Performance**: Progressive image/video loading, intersection observers
- **Deployment**: Vercel with automatic deployments

## Architecture

### CSS Organization
- Modular CSS architecture with component-based styling
- CSS custom properties for consistent theming
- Responsive design patterns with mobile-first approach

### JavaScript Features
- Progressive video loading system with static frame fallbacks
- Interactive portfolio filtering and modal system
- Performance mode toggle for lower-powered devices
- Optimized image loading with intersection observers

### Asset Management
- Videos hosted on GitHub Releases as CDN
- Optimized image formats and lazy loading
- Efficient caching and loading strategies
